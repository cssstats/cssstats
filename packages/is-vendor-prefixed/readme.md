# is-vendor-prefixed

Check whether a property is vendor prefixed.

## Installation

```bash
npm install --save is-vendor-prefixed
```

## Usage

```javascript
const isVendorPrefixed('is-vendor-prefixed')

isVendorPrefixed('-webkit-flex-basis')  // => true
isVendorPrefixed('flex-basis')          // => false
```
